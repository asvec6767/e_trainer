# Компоненты

Эта папка содержит переиспользуемые Vue компоненты.

## Структура

```
components/
├── layout/          # Компоненты макета (Header, Footer, Sidebar и т.д.)
├── common/          # Общие переиспользуемые компоненты
└── ui/              # UI компоненты (кнопки, формы, модальные окна и т.д.)
```

## Примеры использования

### Layout компоненты
- `Header.vue` - Шапка сайта
- `Footer.vue` - Подвал сайта

### Common компоненты
Создавайте здесь компоненты, которые используются в нескольких местах приложения.

### UI компоненты
Создавайте здесь кастомные UI компоненты, которые дополняют PrimeVue.
